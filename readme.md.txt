## 🧭 Navigation Guide

> ℹ️ This application uses modular components with dynamic rendering.

### 📦 Component Navigation

- **To Open a Module or Component**  
  Double-click (or double-tap) the component/module name to view its content in the app.

### 👩‍💼 Employee Types Usage

1. Click the **“+ Add Type”** button to open the form.
2. Fill in the **employee type name** (e.g., `Intern`, `Contract`).
3. Press **Add** to create it.

> ✏️ **To edit/delete an existing employee type**:
- Click the **Edit** button next to the type.
- The same form will open.
- Modify the name and click **Update**.

⚠️ You must open the **Add Type** form first to access the edit functionality.
